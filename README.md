# Ikea Peg Board Attatchments for Skadis system

## Purpose
When first setting up my Skadis system, I chose to orient the panels horizontally rather than vertically. This library contains attatchments 
I have designed for my tools and accsessories to hang on this pegboard, as non of the standard attatchments work well in the chosen orientation.

## Structure
The prototype folder holds previous versions of connectors with slightly different dimensions. Some files in this directory were deemed too tight 
of a fit for my liking, but may prove good for another user. The final folder holds final models out of the prototype stage and in use.

## The Multipeg
The multipeg is my attempt at a universal base peg designed to print reliably on most printers. This model is ideal if you wish to add unique components. 
Import the STL and begin modelling your attatchment using the inner hooks as a starting point. The inner hooks can be printed succesfully both on their side, or upright.
